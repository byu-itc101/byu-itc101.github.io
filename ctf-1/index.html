CTF #1: Introduction to Capture the Flag

Purpose

In this assignment you’ll learn what CTFs are, how flags work, and how to use picoCTF and its web shell to solve a simple challenge.

What is a flag?

A flag is a piece of text that proves you solved a challenge. In picoCTF, flags look like picoCTF{...}.

Rules

  • Spend at least 15 minutes on each challenge before using hints.
  • You can use the internet or AI tools, but use them to learn concepts, not to just get the answer.
  • Take ownership of your learning and experiment.

Example: First Grep on picoCTF

In this example you’ll log in to picoCTF, open the First Grep challenge, download the file using the web shell, and search it with grep to find the flag.

Turning in CTF #1

When you complete the challenge, submit the flag in picoCTF and take a screenshot showing your username, the challenge name, and the completed icon. Paste that screenshot into the CTF #1 assignment on Learning Suite.

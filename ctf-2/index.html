CTF #2 – Linux

Overview

Complete the following picoCTF challenges to practice Linux command line skills. Capture the flag for each challenge below:

  • Magikarp Ground Mission
  • Obedient Cat
  • First Find
  • Plumbing

Resources

  • PicoPrimer: The Shell
  • Command line cheat sheet: https://www.git-tower.com/blog/command-line-cheat-sheet/

Magikarp Ground Mission

Use SSH (Secure Shell) to log in to the provided remote host and navigate the file system to collect all parts of the flag.

  • SSH format: ssh remote_username@remote_host.
  • Once connected, use ls, cd, and cat to explore directories and read files that contain the flag pieces.
  • Keep track of each segment you find; the final flag is assembled from multiple parts.

Obedient Cat

The title hints at the command you need. Download or view the provided file and print its contents to reveal the flag.

  • Use cat filename to display a file’s contents.
  • If the file arrives as a download, save it and run cat locally in the webshell.

First Find

Practice fetching a file and locating hidden items in a directory to uncover the flag.

  • Download with wget <link> inside the webshell to grab the file.
  • Show hidden files by extending ls, for example ls -a or ls -la.
  • Use find or grep as needed to search through directories for the flag text.

Plumbing

Connect to the challenge service and use pipes to capture the streamed flag output.

  • Connect with netcat: nc jupiter.challenges.picoctf.org 7480.
  • Pipes (|) send the output of one command into another; for example, nc ... | tee output.txt saves the data while showing it.
  • Read what the service returns and extract the flag from the streamed text.

IT&C 101 - Intro to Cybersecurity

Purpose

As an IT&C student, Linux is fundamental to your cybersecurity career. Linux is the most used operating system in business, as it is very flexible to each business' needs.

We try to teach you the basics in this class, but you will learn way more going on in this program and your career.

This CTF uses OverTheWire, a free website that teaches different cybersecurity fundamentals with different wargames. The wargame we will be using is Bandit, one designed to teach Linux commands.

But Windows computers will need to do some setup first. Mac users should be able to run Linux commands natively. If you use a Mac or already have WSL, skip to the Accessing the Lab section.

Windows Users: WSL Setup

*** WINDOWS USERS PAY ATTENTION ***

You natively do not have Linux on your computers, but you will when you follow these instructions to download WSL (Windows Subsystem for Linux). WSL creates a Linux filesystem and a place to run Linux commands in your computer!

  1. If your computer is not fully up to date with the latest version of Windows, do that now. WSL can brick if you are not up to date when you download it. Your computer should restart after this update. If you are up to date, move to step 2.
  2. Now, open a Powershell terminal as administrator. Go into your app explorer and type Powershell. Click Run as Administrator.
  1. In your terminal, type wsl --install and Hit Enter.
  1. Your computer should install WSL and restart.
  2. Go back to the app explorer and type WSL.
  1. Click on it, and you should be able to run Linux commands in the terminal! (By default, it is an Ubuntu distribution of Linux)
  2. It should look something like this:

You may be prompted to create a sudo password. Sudo stands for super user do and will allow you higher privileges to do more. CREATE A PASSWORD YOU WILL REMEMBER!!!!

WSL install instructions (Microsoft): https://learn.microsoft.com/en-us/windows/wsl/install

SSH into Bandit

Now that you can use Linux commands, let's use SSH to connect to Bandit. SSH stands for Secure Shell and allows us to login to remote computers.

They give us:

  • A username: bandit0
  • A host: bandit.labs.overthewire.org
  • A port (place where the process is running): 2220
  • And a password: bandit0

The command to connect is:

ssh bandit0@bandit.labs.overthewire.org -p 2220

After SSH command, place to put password. When entering a password in a terminal, it will not be visible, but it will be there when you type. Press Enter to login after password attempt.

As you complete levels, the username will change to correspond to the level you are connecting to. Going to level 1 -> bandit1, Going to level 2 -> bandit2 and so forth. The host and port will not change. The thing that changes the most is the password.

Objective

The point of each level is to find the password to the next level. The password is a random set of characters and numbers.

Example of a Password: 5ABMh1JI91bUIZZPXDqGanal4xvAg8J9

You will use the internet and your knowledge of Linux commands to solve these challenges.

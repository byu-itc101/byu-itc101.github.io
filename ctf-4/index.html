CTF #4: OSINT Tools

IT&C 101 - Intro to Cybersecurity

Purpose

Open-Source Intelligence (OSINT) is widely used in the real world to gather information from publicly available sources such as social media, news articles, government reports, and websites. Penetration Tests start with OSINT about a company, seeing if they can find any information about the company, workers, or maybe information about security! Law enforcement agencies use OSINT to track criminal activity, investigate fraud, and locate suspects. You have probably used OSINT to “stalk” people’s Instagram! This CTF is designed to teach you different real OSINT tools, such as the Wayback Machine so that you can (ETHICALLY) use them in other CTFs, real world engagements, or for fun!

WayBack Machine

The Way Back Machine is a tool built by the Internet Archive that takes snapshots of websites as they once were for safekeeping, research, journalism etc. It preserves the history of the Internet. We will be using it to learn information about websites from a different time. Answer these questions about various websites; there should be only one answer for each.

https://web.archive.org/

Questions

  1. Going back to March 2000 on www.iana.org, when exactly was the site last updated? Answer with the date in the footer, eg 11-March-99
  2. According to wikipedia.com snapshot taken on February 9, 2003, how many articles were they already working on in the English version?
  3. Going back to November 1998 on google.com, what address hosted the non-alpha "Google Search Engine Prototype" of Google? Answer with the full address, eg http://google.com
  4. According to the paypal.com website in October 1999, what could you use to "beam money to anyone"? Answer with the product name.

Google Dorking

Google Dorking is a powerful feature in Google that allows people to focus their Google searches using keywords and wildcards such as site, filetype, inurl, AND, *, “quote” etc. This section is dedicated to getting you familiar with the power of Google Dorking. Provided below are some cheat sheets with Google Dorking keywords.

Questions

  2. Our fellow cyber student Cameron created a wordpress.com website while in the Czech Republic! Use Google Dorking to find the website that contains the flag. The flag is in the format byuctf{}.

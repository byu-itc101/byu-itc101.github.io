Caesar Cipher

Change each letter of the alphabet to another a fixed number of places away.

Vigenere Cipher

A method of encrypting alphabetic text where each letter of the plaintext is encoded with a different Caesar cipher, determined by the corresponding letter of a key.

For example, if the plaintext is “attacking tonight” and the key is oculorhinolaringology, then the first letter a of the plaintext is shifted by 14 positions in the alphabet (because the first letter O of the key is the 14th letter of the alphabet, counting from zero), yielding o; the second letter t is shifted by 2 (because the second letter C of the key is the 2nd letter of the alphabet, counting from zero) yielding v; the third letter t is shifted by 20 (U) yielding n, with wrap-around;

Kryptos: an encoded message at CIA headquarters. The author used Vigenere ciphers.

Steganography

Steganography: concealing info in other files within other non-secret data.

Data within data: data-ception.

SQL Injection

SQL Injection: code injection technique that allows hackers to access and manipulate data.

When you provide login credentials, code is run to look up if your username and password exist and if the username and password match.

Hacker will put in malicious code in login credentials that can ask and retrieve data from the database!!!

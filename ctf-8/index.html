Cookies!!

  • Cookies are found in the cookie jar (Inspect -> Application -> Cookies -> Cookie Jar)
  • Cookie create state for HTTP/S (personalization, authentication, other things)
  • You can manipulate cookies, since they are on your machine.

Websites are Files??

  • Websites are just a bunch of files we request from a server somewhere.
  • The address bar is essentially a file management tool, and we can access directories and files with it!
  • directory/ - with a slash you move into a directory
  • File - no slash, you will move into a file

Robots.txt

  • Robots.txt is a special file in the root directory of a website and can stop indexing of certain files
  • What indexing means is who can access the files of the website
  • We stop indexing from certain URLs (file crawlers) to prevent overloading the website
  • It can have important information … so if we can see it that is a vulnerability

logon

Cookies are used for authentication. What cookies are there in the cookie jar? Can we change it to a certain value to login?

Secrets

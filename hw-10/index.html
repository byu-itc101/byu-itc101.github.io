Homework 10 – Wireshark Exploration

IT&C 101 - Intro to Cybersecurity

Submission template: assets/HW10.docx

Purpose

Networks are foundational to cybersecurity. This homework introduces packet capture and packet analysis using Wireshark so you can understand how network data moves and what protocol-level evidence looks like.

Download and Install

Download Wireshark: https://www.wireshark.org/download.html

  • Windows: download the x64 installer.
  • Mac: M1/M2 should use ARM; older Macs should use Intel.
  • Use default installer settings.

Objective

Create a packet capture (.pcap) and investigate packet contents to build understanding of network protocols and encrypted traffic.

Disclaimer: Use Wireshark ethically. Do not capture non-consenting parties' traffic or use packet analysis for unauthorized activity.

Procedure

Packet Capture

  1. Open Wireshark.
  2. Double-click Wi-Fi (or your active interface) to start capture.
  3. Open a browser and visit byu.edu.
  4. Return to Wireshark and stop the capture.

Investigation

Click a packet and inspect packet list fields plus the Packet Details and Packet Bytes panes.

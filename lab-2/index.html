Lab 2 – Linux CLI

Submission template: assets/Lab2.docx

Purpose

As an IT&C student, you will need to be familiar with the command line, which you will use extensively throughout your time at BYU and career. This lab is to help teach you principles on using the command line and other utilities available via the command line such as “vim” (a document editor). For this lab, much of your education will be self-driven (you won’t get much out of it doing the bare minimum). Professors and employers alike will encourage you to learn things on your own.

Objective

Learn about and apply the Linux command line and vim by completing various CLI exercises.

Procedure

Linux

  • Go to Linux Survival at https://linuxsurvival.com/linux-tutorial-introduction/.
  • Read the information about the Linux Command line.
  • Take all 4 quizzes. Include screenshots of the 4 quizzes in your submission.

Vim

  • Go to https://www.vim-hero.com/
  • Do the challenges of navigating through Vim
  • Take screenshots of your results from 3 challenges (they should show speed, accuracy, proficiency)
  • Try the bug review game at the end of the Basic Vim section. You do not have to fully complete it, but it is a fun review!

Submission

Submit a single Word docx file or pdf file that includes screenshots of showing your completion along with answers to the following questions. You can copy and paste these questions and enter your responses after each one:
